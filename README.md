# يلا نسوالف - Yalla Nsuwalef

تطبيق دردشة عربي مع دعم الغرف والمكالمات والشحن

## الميزات
- تسجيل الدخول برقم الهاتف أو Google
- إنشاء والانضمام إلى غرف الدردشة
- دفعع جميع الدول العربية
- نظام شحن ومحفظة إلكترونية
- إرسال الهداي
- دعم الصوت والفيديو (قريباً)
- واجهة عربية كاملة

## التقنيات المستخدمة
- **Backend**: Node.js, Express, Socket.io, MongoDB
- **Frontend**: Flutter (Android)
- **Database**: MongoDB
- **Realtime**: Socket.io
- **Auth**: JWT, Firebase Auth

## الإعداد للتطوير

### المتطلبات
- Node.js 18+
- MongoDB
- Flutter SDK

### تشغيل Backend
```bash
cd backend
npm install
cp .env.example .env  # عدل المتغيرات
npm run dev
```

### تشغيل Frontend
```bash
cd mobile
flutter pub get
flutter run
```
